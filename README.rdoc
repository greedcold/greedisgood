== README
Проект предназначен для ведения домашней бухгалтерии.
Пользователь будет составлять список своих затрат за определенный период времени и сравнивать сумму этих затрат с прибылью, получаемую за тот же временной период.
Также планируется создание некоторых целей, требующих материальных затрат


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
